Amapiano Stream – Site Plan

Site Name

Name: Amapiano Stream

Why this name? It is short, easy to remember, and clearly shows the purpose — streaming Amapiano music.

Possible Domain: amapianostream.co.za

Site Purpose

This website will allow users to stream Amapiano music, discover new artists, and learn about trending songs. The site will also provide artist profiles, playlists, subscription options, and music recommendations.

Scenarios (User Questions)

  • How can I discover new Amapiano artists?
  • Can I listen to playlists for free?

Color Schema

The colors below will be used for the website:

  • #1A1A2E – Dark Background
  • #FFCA28 – Accent Color / Buttons
  • #FFFFFF – Text on dark areas

Typography

Selected Google Fonts:

  • Poppins (Headings)
  • Roboto (Body Text)
